# DNS 101.6.6.6

TUNA 现提供一台域名解析服务器，进行**准确**的递归解析服务：

    101.6.6.6

服务器位于清华校内，提供 IPv4 服务。在使用有线网络时，校内师生可以使用这一服务进行域名解析，而无需进行校园网认证。

请将系统网络设置中 DNS 项修改为 `101.6.6.6`，即可开始使用。具体配置方式暂缺，请[帮助我们](https://github.com/tuna/mirror-web/)补充文档。
